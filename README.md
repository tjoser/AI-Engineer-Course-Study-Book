# AI Engineer Course Study Book

A structured study book for learning AI engineering from a software-engineering perspective.

This repository is based on the public topic structure of the YouTube playlist **The AI Engineer Course 2025** and expanded with original explanations, exercises, interview questions, and project ideas.

> Important: this is **not** a verbatim transcript dump. The goal is to create a legal, study-friendly, original companion book that covers the concepts deeply without copying copyrighted course transcripts word-for-word.

## How to use this book

1. Read the chapters in order.
2. After every chapter, answer the review questions.
3. Build the small practice tasks.
4. Finish with one of the capstone projects.
5. Keep notes in your own words.

## Recommended learning outcome

By the end, you should be able to:

- Explain the difference between AI, ML, deep learning, NLP, generative AI, and LLMs.
- Build an LLM-powered backend API.
- Create a RAG chatbot over documents.
- Use embeddings and vector databases.
- Understand LangChain-style application architecture.
- Evaluate and improve AI application quality.
- Prepare portfolio projects for AI/software engineering roles.
